Backup & Restore

Your companions, personas, memory books, and settings live only in this browser — clearing it loses them. Save a backup to a file on your computer, then Import to restore. API keys are NOT included — keep those saved somewhere safe on your own.

Tools

Let the model run: calculator, web_search (Brave), url_fetch, file_read / file_search / file_write (in a folder you choose), and image_generate. Works with every tool-capable provider.

Get a free key at brave.com/search/api. Stored in browser localStorage, sent to the local Python server per request.

Switching companions? Point this at their sandbox folder before chatting.

file_read is scoped to this folder. Paths starting with .. or escaping this folder are rejected.

Used by the image_generate tool. Separate from the OpenAI chat key. Stored in browser localStorage, sent to the local Python server per request.

Auto-Memory (Heartbeat)

When you go quiet, your companion takes one quiet background turn to tidy up — reviewing the chat, updating its Desk Notes file, and saving durable memories on its own. Needs tool use enabled on a tool-capable provider. Off by default; it spends a few tokens each time it fires.

How long you have to be idle before the heartbeat runs. It fires once per quiet gap, not on a loop.
